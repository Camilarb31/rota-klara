# 🚚 Rota Klara

Sistema web de gestão e otimização logística desenvolvido para a **Águas Klaras Distribuidora LTDA** — Capivari/SP.

## 📋 Sobre o Projeto

A Rota Klara automatiza o fluxo de entregas de água mineral e gás de cozinha, eliminando o planejamento manual de rotas e centralizando o controle operacional em uma plataforma acessível.

## ✨ Funcionalidades

- 🔐 Autenticação com perfis (Administrador / Entregador)
- 📊 Dashboard com visão em tempo real das operações
- 📦 Gestão completa de pedidos
- 🗺 Roteirização automática de entregas (Nearest Neighbor)
- 👥 Cadastro de clientes com geocodificação de endereços
- 🛵 Acompanhamento de entregadores no mapa
- 📱 Interface simplificada para uso em campo

## 🛠 Tecnologias

- React 18
- Vite
- CSS3 / SVG (mapa interativo)

## 🚀 Como Rodar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```
